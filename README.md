# sebastienbarbier.blog

This repository host [blog.sebastienbarbier.com](https://blog.sebastienbarbier.com). 

## Local installation using virtualenv 

```
$> python3 -m venv apps

$> source apps/bin/activate

$> pip3 install -r requirements.txt
```

## Run locally

```
$> source apps/bin/activate

$> make devserver
```
